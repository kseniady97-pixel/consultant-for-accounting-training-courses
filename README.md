# consultant-for-accounting-training-courses
Созданный с помощью Voiceflow ИИ-Консультант по обучающим курсам на сайте в формате виджета с интеграцией с Google-Таблицей
1. Описание проекта
   
1.1. Цель проекта
Создание веб-сайта для онлайн-школы по бухгалтерскому учету с интеграцией ИИ-консультанта (в формате виджета). Консультант будет предоставлять информацию о курсах, помогать в их выборе.

1.2. Используемые технологии
- Voiceflow;
- Creatium;
- API платформы OpenAI;
- Google Таблицы.
2. Основные функции бота

2.1. Основные функции
- Ответы на часто задаваемые вопросы по курсам. 
- Рекомендации по выбору курса на основе интересов пользователя. 
- Сбор заявок на курсы и передача информации по ним в Google Таблицу.

2.2. Ключевые технические детали. Архитектура 
Инструменты:
 - Платформа Voiceflow для создания и кастомизации ИИ-консультанта. 
- Конструктор веб-сайтов Creatium для создания и интеграции виджета. 
- API платформы OpenAI для подключения ассистента. 
Подсистемы: 
- Интерактивный виджет ИИ-консультанта. 
- Каталог курсов.
- База данных о курсах в формате json.
3. Реализация проекта

3.1. Этапы работы
1) Генерация наполнения сайта, перечня предлагаемых курсов.
2) Создание шаблона в Voiceflow, настройка ассистента, подключение интеграции с Google Таблицей.
2) Разработка сайта с использование Creatium. 
3) Вставка виджета ассистента. 
4) Проверка работоспособности ассистента в браузере. 
5) Контрольное тестирование функциональности виджета.

3.2. Итоговые результаты
В Voiceflow была реализована следующая структура: после получения приветственного сообщения пользователь начинает взаимодействовать с ИИ-консультантом, роль которого на основе выявленного опыта и предпочтений пользователя - посоветовать конкретный курс из перечня существующих и после подтверждения со стороны пользователя - перевести на блок с заявкой. В ходе последовательных блоков с разветвлением собирается информация о пользователе (имя, телефон, выбранный курс) и передается в Google таблицу с указанием даты обращения.
Сайт состоит из блоков информации о компании, курсах, отзывах и контактных данных для связи. Взаимодействие с ИИ-консультантом - через - виджет.
